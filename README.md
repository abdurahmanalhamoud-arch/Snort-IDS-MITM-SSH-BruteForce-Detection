# Intrusion Detection System Project – Snort IDS

## Project Overview
This project demonstrates the implementation of a Network Intrusion Detection System (IDS) using Snort to detect cyber attacks in a simulated network environment.

The project includes attack simulation, network traffic analysis, and custom Snort rule creation to detect specific attacks such as Man-in-the-Middle (MITM) and SSH Brute Force attacks.

---

## Attacks Simulated

### 1. Man-in-the-Middle (MITM) Attack
- Tool used: Bettercap
- Network discovery and target identification
- ARP Spoofing attack
- Traffic interception
- Packet analysis using Wireshark
- Detection using Snort rules

### 2. SSH Brute Force Attack
- Tools used: Nmap, Metasploit
- Port scanning to discover SSH service
- Brute force login attack using Metasploit
- Detection using custom Snort rules
- Alert generation when attack detected

---

## Detection Method
Snort IDS was configured to monitor network traffic and detect attacks using custom rules.

Custom Snort rules were created to detect:
- MITM attack
- SSH Brute Force attack

When an attack occurs, Snort generates alerts indicating suspicious or malicious activity.
